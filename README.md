# text-moderation-ml-service
Интеллектуальный сервис автоматизированной модерации текстового контента на основе трансформерной нейросети (RuBERT-tiny2).  Реализованы загрузка данных, обучение модели, оценка качества, GUI и инференс. Проект выполнен в рамках ВКР в ЧОУ ВО "Московский университет им. С.Ю. Витте".
